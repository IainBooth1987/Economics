Game Configuration
Game Theory Concepts In This Simulation

Incomplete Information

Faithfuls cannot observe traitor types. They must infer from observable actions — the core challenge of this game.

Bayesian Updating

Each event (murder, vote, speech) provides evidence. Rational players update P(Traitor|Evidence) using Bayes' rule.

Signalling Theory

Accusations and defences are signals. Without cost, they are cheap talk. Costly signals are more credible — harder to fake.

Mechanism Design

Voting rules shape equilibrium behaviour. Plurality vs. majority vs. supermajority create different incentives for coordination and defection.

Prisoner's Dilemma

At the Round Table, each Faithful must decide whether to cooperate (coordinate on a target) or defect (protect allies). Individual rationality can undermine collective goals.

Repeated Games

Because the game runs for multiple rounds, reputation matters. Consistent voting patterns accumulate as evidence — a key insight from Folk Theorem.
